# Pascal Programming Exercises

Pascal programs from my university's programming course.

## Project Structure
- `src/`: Source code for the exercises (`.pas` files).
- `bin/`: Compiled executables.
- `.vscode/`: Task and environment configuration.

## How to Compile
This project uses the Free Pascal Compiler (FPC). You can compile the exercises using the configured VS Code build task:
`Ctrl + Shift + B`
